
# Chatbot para responder preguntas

El siguiente proyecto consiste en la elaboraciÃ³n de un prototipo de Agente Conversaional basado en inteligencia artificial para responder las preguntas mas frecuentes sobre los estudios de cuarto nivel en la Universidad Nacional de Loja. Se utiliza el modelo de lenguaje DistilBERT como red neuronal que se encarga de llevar a cabo las prediciones en base a las preguntas que son ingresadas por los estudiantes.


## Tech Stack

**Client:** React, HTML, CSS, TypeScript, TailwindCSS, Vite

**Server:** Python, DJango, Pytorch

**Adicionales** Git, GitHub, Google Colab, DistilBERT,
